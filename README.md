# Easy JSON2JSON
    
## Description
A python package for easy date use.

# Install 
    pip install easydate

# Usage example 

## Case #1 Simple JSON
```python

from easydate.easydate import *

result = str_to_date("22/03/2020", input_format="%d/%m/%Y")

print(result)

# 2020-03-22

```
